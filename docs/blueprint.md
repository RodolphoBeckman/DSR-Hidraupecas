# **App Name**: BudgetEase

## Core Features:

- Client Registry: Register and manage customer data, including contact information and service history.
- Salesperson Registry: Register and manage salesperson data.
- Payment Plan Registration: Configure and manage different payment plans to offer clients.
- Payment Plan Management: Create, view, update, and delete various payment plans for flexibility.
- Dynamic Budget Creation: Create a new budget by entering the service description and its corresponding value, accumulating them in real time.
- QR Code Integration: Incorporate a QR code for Pix payments, set up in the configuration, and visible only in the PDF budget.
- WhatsApp Sharing: Generate a PDF budget and redirect to WhatsApp with the budget attached for easy sending.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), representing trust and professionalism in financial services.
- Background color: Light gray (#F9FAFB), for a clean, modern and professional look.
- Accent color: Soft lavender (#A78BFA), to highlight key actions and information with a touch of sophistication.
- Body and headline font: 'Inter', a versatile sans-serif for clear, readable text in both titles and content.
- Use minimalist icons to represent services, plans, and tools for intuitive navigation.
- Design a clear layout with distinct sections for forms, budget details, and payment options, optimizing readability and user flow.
- Apply subtle transition effects and loading animations to enhance the app's responsiveness.
